# The Quality Time Co.

A cinematic, minimal Next.js 14 application.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript**
- **Tailwind CSS**
- **Google Fonts**: Space Grotesk, Inter, JetBrains Mono

## Design System

| Token | Color | Usage |
|-------|-------|-------|
| Background | `#1C2127` | Deep slate background |
| Text Primary | `#F7F3ED` | Cream - headings & primary text |
| Text Secondary | `#B4C7C2` | Sage - secondary text |
| Accent | `#FF5D5D` | Coral - buttons & highlights |

### Typography

- **Headings**: Space Grotesk (bold, -0.02em tracking)
- **Body**: Inter
- **Numbers/Code**: JetBrains Mono

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
